# 🎓 Student Management System (Angular)

A full-featured Student Management System built using Angular with CRUD operations and modern UI.

---

## 🚀 Features

- ✅ Add Student
- ✏️ Update Student
- ❌ Delete Student (with confirmation)
- 📋 View Students in table format
- 🔄 Real-time data refresh
- ⏳ Loading spinner
- 🔔 Toast notifications
- 🎨 Responsive UI using Bootstrap

---

## 🛠️ Tech Stack

- Angular (Standalone)
- TypeScript
- Bootstrap
- REST API (Spring Boot / Backend)
- Git & GitHub
